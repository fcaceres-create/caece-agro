# AgroSmart — Presentación (esqueleto)

> Esqueleto de la defensa oral. Cada bullet es un slide tentativo.

1. **Portada y problema**
   - Decisión agronómica en Argentina: aptitud, rendimiento, riesgo.
2. **Propuesta**
   - Sistema híbrido Python + SWI-Prolog.
   - Cobertura: 11 cultivos extensivos, 5 regiones agrícolas.
3. **Datos**
   - Fuentes: MAGyP, Open-Meteo, SoilGrids, INTA/FAO.
   - Construcción del dataset maestro georreferenciado.
4. **Capa estadística (Fase II)**
   - EDA, hallazgos, modelo de regresión Random Forest por cultivo.
5. **Capa simbólica (Fase III)**
   - Hechos generados desde el EDA + hechos expertos.
   - Reglas de aptitud, riesgo y recomendación.
6. **Aporte propio (Fase IV)**
   - Datos -> Conocimiento (percentiles -> hechos Prolog).
   - ML <-> Lógica (validación cruzada de la predicción).
7. **Demo en vivo**
   - Consulta de un lote, salida del integrador.
8. **Resultados, limitaciones y trabajo futuro.**
9. **Cierre y preguntas.**
